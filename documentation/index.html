Installation

In order to install our template do the following:

  1. Unzip the jayne folder.
  2. Pick the HTML template that you would like to use (You can remove the other ones).
  3. Optionally you can remove unnessary files that you don't want to use. For example SCSS folder that is located in assets folder (if you don't plan to cusomize style of your template), files from the color-themes folder that is located in css folder (you can pick one skin that you want to use and remove the other ones).
  4. Upload files through the FTP.

Customizing styles

The main stylesheet style.css is located in the assets/css folder.

In the assets/css/color-themes folder there are 8 stylesheets for different color skins. By defaut each HTML template has the blue color skin (you can see it is included after the style.css stylesheet in the HTML template). In order to change the color skin for you template just change the name blue in the <link id="color-stylesheet" href="assets/css/color-themes/blue.css" rel="stylesheet"> to the name of a color skin that you want to use.

Customizing fonts

Stylesheet fonts.css has the font faces for other fonts if you don't like the one that is used by default. In order to use them either include fonts.css in the HTML template before style.css stylesheet and change the font in scss/bootstrap/_variables.scss or just copy the font face of the font you like from font.css to style.css stylesheet and change the font manually.

Images and logo

All of the images are located in the assets/images folder. To change the logo either change the name (title) or delete <h1 class="b-logo__title">Jayne</h1> from the HTML template and upload an image logo.png into the assets/images/logo folder. In order to change the background pictures for the light background slideshow replace the images in the assets/images/light folder. To change the background pictures for the dark background slideshow replace the images in the assets/images/dark folder. Static background images are assets/images/light/image1.png for the light image and assets/images/dark/image1.png for the dark image. In order to change the images for the slider replace the pictures in the assets/images/slider folder. To change the picture of a product replace the image in the assets/images/product folder.

Twitter widget

To change the twitter account displayed in the twitter widged you need to:

  1. Create an official twitter widget by going to this link
  2. Go to the end of the HTML template that you use and replace username "docothemes" to the username of your twitter account
  3. go to the twitter/index.php file and replace consumer key, consumer secret key, user token, user secret key with the keys you acquire by creating an official twitter widget

Contact form and newsletter

To receive the emails that user inputs in the newsletter form you would need to replace variable $to = "email@gmail.com"; with your email address. Also, all of the emails received from users are recorded to text-files/emails.txt file.

If you are using the landing page HTML template then in order to receive contact forms to your email address replace variables $to = "email@gmail.com"; and $from = "email@gmail.com"; with your email address.

Map

If you are using the HTML template that has map as a background you would need to:

  1. Go to the assets/js/big-map.js file
  2. Edit gm_address_latitude and gm_address_longitude variable to the latitude and longitude of you address (you can find them here)
  3. You can change color of the map and the map marker by editing gm_landscape_color, gm_water_color, gm_circle_color variables
  4. Optionally you can change the location of the map marker by editing gm_offset_x and gm_offset_y variables

If you plan to use the landing page HTML template you can edit the contact map by:

  1. Edit gm_address_latitude and gm_address_longitude variables to the latitude and longitude of you address (you can find them here)
  2. Change the color of the map marker by editing gm_circle_color variable.
